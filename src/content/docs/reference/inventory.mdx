## Family 1 - rBot / rxBot (121 dirs, ~93 unique)

The largest and most influential IRC botnet codebase in the collection. Originated March 2004 by Nils, D3ADLiN3, Edge, bohika, rewe; later developed by RacerX90 into rxBot. C++ on Win32, IRC-based C&C.

**Aggregated capabilities:** DCOM/LSASS/ASN.1/NetAPI/PnP/Symantec exploits · SYN/TCP/ICMP/UDP/ACK DDoS · MSN/AIM/Yahoo/USB spreading · keylogger · packet sniffer · SOCKS4/5 proxy · CD key theft · AV killer · registry persistence.

## Family 2 - sdBot / spyBot (73 dirs, ~42 unique)

The foundational IRC bot by author \[sd\], from which rBot itself was derived. Single-file C source originally compiled with LCC. Spans sdbot 0.4b through spyBot 1.4.

**Aggregated capabilities:** Basic IRC C&C · SYN/UDP floods · NetBIOS brute · keylogger (spyBot) · HTTP download server (spyBot) · email spread · RAR infection.
